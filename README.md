# MySQL CRUD Operations using Python

##  Project Description
This project demonstrates CRUD (Create, Read, Update, Delete)** operations using Python and MySQL.  
It connects Python programs to a MySQL database and performs database operations on tables.
The project is suitable for beginners who want to learn Python-MySQL connectivity and basic database handling.

##  Technologies Used
 Python 3
 MySQL
 MySQL Connector for Python

##  Database Details
  Database Name: company_db
  Tables Used:
   tbl_dept
   tbl_emp

## Project Files Description

connect_db_25.py  
Establishes a connection with the MySQL database.

create_table.py  
Creates the required tables in the database.

Add_record.py  
Inserts new records into the database tables.

Display_Records.py  
Displays records from the database.

display_records_join.py  
Retrieves records using JOIN queries.

update_record.py  
Updates existing records in the database.

delete_record.py  
Deletes records from the database.

## CRUD Operations
###  Create (Insert Records)
 Add department details
 Add employee details

###  Read (Display Records)
 Display all department records
 Display employee records using JOIN queries

###  Update (Modify Records)
 Update department name and location

###  Delete (Remove Records)
 Delete department records using department ID
